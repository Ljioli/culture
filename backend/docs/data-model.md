# WeCulture 数据结构与后台功能映射

本文件根据现有云函数 `model/controller/service` 和小程序后台页面反向整理。集合统一使用 `bx_` 前缀。

| 集合 | 主要用途 | Web 后台功能 |
| --- | --- | --- |
| `bx_admin` | 管理员账号、MD5 密码、角色、状态、登录 Token | 登录、管理员增删改、启停、修改密码 |
| `bx_log` | 管理员操作日志 | 日志查询、清空 |
| `bx_user` | 小程序用户、昵称、手机、头像、状态、自定义资料 | 用户列表、详情、审核、禁用、删除 |
| `bx_news` | 公告和服务内容、分类、正文、封面、排序、推荐 | 新增、编辑、删除、上下架、推荐 |
| `bx_product` | 线路/吃喝/住宿/购物攻略和自定义内容 | 新增、编辑、删除、上下架、推荐 |
| `bx_activity` | 活动时间、地址、人数、审核与取消规则 | 新增、编辑、删除、上下架、推荐 |
| `bx_activity_join` | 活动报名、审核状态、核验码、签到状态 | 名单查询、审核、签到、删除 |
| `bx_info` | 用户游记、分类、正文、状态、推荐 | 内容审核、状态、推荐、删除 |
| `bx_comment` | 公告/攻略/活动/游记评论 | 随主体数据关联维护 |
| `bx_fav` | 用户收藏 | 随用户与主体数据关联维护 |
| `bx_history` | 用户浏览历史 | 用户详情、历史清理 |
| `bx_setup` | 关于我们、首页推荐等系统内容 | 关于我们、小程序二维码、系统设置 |

## 暂不处理

智能导览使用独立 Flask + Ollama + SQLite/Chroma 演示服务，本期不纳入 CloudBase Web 后台。
